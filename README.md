# 📘 MongoDB Basics Notebook

This repository contains a Jupyter Notebook that introduces the fundamentals of MongoDB, a popular NoSQL database. It is designed for beginners who want to understand how MongoDB works and how to interact with it using basic commands.

---

## 🚀 Contents

The notebook covers:

- Introduction to MongoDB
- NoSQL vs SQL databases
- Installing and setting up MongoDB
- Connecting to MongoDB
- Basic CRUD operations:
  - Create (Insert documents)
  - Read (Query documents)
  - Update (Modify documents)
  - Delete (Remove documents)
- Working with collections and databases
- Basic query operators

---

## 🛠️ Requirements

Before running the notebook, make sure you have:

- Python (>= 3.x)
- Jupyter Notebook or Jupyter Lab
- MongoDB installed locally OR MongoDB Atlas account

  pip install pymongo
